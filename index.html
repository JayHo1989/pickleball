📲 Cài App & Backup
🤖 Android (Chrome)
1. Mở file này bằng Chrome
2. Bấm menu ⋮ góc trên phải
3. Chọn "Thêm vào màn hình chính"
4. Bấm Thêm → Icon app xuất hiện ✅
🍎 iPhone / iPad (Safari)
1. Mở file trong Safari (bắt buộc)
2. Bấm nút ⎙ Chia sẻ thanh dưới
3. Kéo xuống → "Thêm vào Màn hình chính"
4. Bấm Thêm → Icon app xuất hiện ✅
💾 Dữ liệu lưu thế nào?
• Tự lưu vào bộ nhớ điện thoại sau mỗi thao tác
• Tắt app, tắt máy → mở lại vẫn còn nguyên
• Không cần internet sau lần tải đầu tiên
